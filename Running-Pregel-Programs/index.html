Running Pregel+ Programs

发表于 2017-02-27

Download

  • Download system code and extract to $HOME/pregelplus

  • Download application code of Hash-Min and extract to $HOME/hashmin

  • Download toy graph and put it onto HDFS

    hadoop fs -mkdir /toyFolder

    hadoop fs -put toy.txt /toyFolder

Compilation

  • Download makefile sample and put it under $HOME/hashmin

  • replace the system code directory of the makefile

  • compile the source code to the binary file run

    make -f Makefile.txt

Running

mpiexec -n number-of-processes -f process-to-machine-mapping-file compiled-binary-file other-arguments
