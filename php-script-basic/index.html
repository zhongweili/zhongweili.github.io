引用其他文件

很多情况下，需要调用其他PHP文件里的类和函数，有两种方式引用：require 和 include。唯一的区别是，引用出错时require会报error，include只会报warning。

require_once 'file';

文件可以是当前路径下的文件名，可以是绝对路径，可以是相对路径。相对路径的写法：

include_once __FILE__.'../../dir/file';
