目录结构

  • / root directory
  • /boot linux kernel
  • /etc configurtion files
  • /bin /usr/bin programs for the system
  • /sbin /usr/sbin programs for admin
  • /usr things support user applications
  • /usr/local installation of software
  • /var files change as the system is running
  • /lib shared files(DLLs)
  • /home personal work
  • /root superuser’s home directory
  • /tmp temporary files
  • /dev
  • /proc
  • /media /mnt
