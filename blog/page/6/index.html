一对一

发表于 2017-04-17

罗辑思维前几天推出了一个对谈节目—《长谈》，是”得到“创始人罗振宇对锤子科技创始人罗永浩的一次访谈，一共进行了将近9个小时，抛出了200多个问题，宣称是人类电视史上最长的一次访问，怪不得叫”长谈“呢。

我本身对这类访谈形式是有些兴趣的，再加上这个长谈中的两个胖子都很有意思，所以抽时间断断续续听了听。因为访谈是围绕罗永浩这个“相声演员”的创业过程进行的，所以听完后对当代中国创业者的心路历程和精神世界有了更多的认识。

但收获远不止于此。我之所以喜欢访谈节目，很大原因是我喜爱了解人，而近距离的全方位的交流更能看清一个人，更何况这个长谈的问题如此之多。《长谈》的副标题写着“让他把话说完”，罗永浩作为一个影响不小的网红，太容易被媒体过分标签化了，这个节目至少让人可以从其他的角度来认识他，比如他最危险的时候曾个人担保借钱超过一个亿，比如他曾有不小的社交恐惧。罗振宇说《笑林广记》里有个笑话：一个聋子，看见别人放炮仗，然后他就觉得特别奇怪，说好好的一个花纸卷，怎么说散就散了呢？这个笑话的意思是，我们观察一个事物，如果丢失了一个维度，比如聋子，那么只看现象永远也不知道发生了什么。

不仅看公众人物是这样，生活中的每个人也一样，看到的只是表象。人是会伪装的，以至于习惯了伪装会把一些东西深深埋起来，没有足够的接触外人根本看不到。所以我的识人之法就是与人一对一的交流。一对一，才能让人有所倾吐，才能观察出东西，也许一个光鲜亮丽的人背后有不得言说的苦衷，也许一个沉默寡言的人在某些方面欢脱又活跃。

一对一，就是能给人这样的意外，常常像是从另一扇窗找到了另一个角度。一段感情关系中的两个人更是如此，初接触时是“我看青山多妩媚，料青山看我亦如是”，走得近了总会在某些地方是“入之愈深，其进愈难，而其见愈奇”，当真是”不识庐山真面目，只缘不在此山中“。因此我常常感到，人实在太复杂而神奇了。所以这也是我不喜欢群体聚会的原因吧，一阵寒暄和几个玩笑，时间就过去了，刚刚交浅，不得言深。

不过并不是所有的一对一都能碰撞出火花，愉快的长谈都需要投机，以及真诚。我在本科时候参加学校项目，做过一个简单至极的手机App，当时移动互联网创业席卷校园，竟有校记者团的一名小记者约我采访，我讲了半天App的功能和设计逻辑，他机械地点着头，像是应付差事。本以为这种采访就这样无聊地结束了，意外的是，等到要求的问题问完，他搬出自己开打印店的经历热情地跟我聊起来，到后面又聊起他对App的看法、对手机的看法，让我觉得这才是一对一的对话，能从中发现很多东西。

更意外的是，过了几天，一个对创业颇有兴趣的同学从校园网看到这次采访稿，专门找到我来了个一对一。我已经不记得那天晚上说了些什么，只记得两个未出茅庐的少年煮酒论英雄，大肆谈论着对互联网对未来的憧憬和妄断，而时间匆匆流过。一对一的谈话，不仅能让人发现一些可能从不曾留意的角度和闪光点，另外一层意义是，谈话中让人有一种难能可贵的感觉：原来你和我一样。

为什么一对一的谈话能够迸发出意外之喜？因为它更难，所以深入。

去年获得诺贝尔文学奖的鲍勃迪伦，低调的他平淡如水，在简短的获奖感言中有这样一段话：

但有件事我必须要说，作为一个表演者，我为5万人演唱过，也为50人演唱过，而为50个人表演难度更高。因为5万人会融成单一人格，但50人不会。他们每个人都是一个鲜明的个体，不同的身份，自成一体。他们能更加清晰地感知事物。你的真诚以及它如何反应出你的才华和深度，都在经受考验。诺奖评委会的人数之少，我是清楚的。

面对一个人表演相比面对成千上万人表演更重要也更难，这个反常识值得铭记，因为现在的社会大环境已经印证了这样的逻辑。为什么在网络直播中出现的不是小品、相声那样的镜头，而是那么明显的脸、那么近的距离？因为主播面对的不是几十万的观众，而是一个观众。要让观众感到，这是你在与主播一对一的互动，就像你与一个真实的人面对面一样。

写文章也是这样，写作者面对的不是所有读者，而是一个一个鲜活的个体。所以我提醒自己，我发的文章不该是广播，而是一封一封的信。

道理是苍白的

发表于 2017-04-04

道理是苍白的

《奇葩说》第四季开播了，这大概是我唯一看的综艺节目了，如果这算是综艺节目的话。

最开始是奔着高晓松和蔡康永去的，结果看了两集就迷上了马东，插广告的功力正是无人可及。后来一集一集看下来，硬是眼睁睁看着上台紧张的新人都变成了自带台风的老奇葩。感觉每一集看过，除了笑得前仰后合还能引发一些思考，让我意识到这个问题原来还可以这样考虑。这样下来，虽然我在平常生活中还是不会好好说话，不过感觉情商还是有所提高的。

《奇葩说》之所以吸引我，更重要的原因是，这是一个价值观在碰撞中熔炼又创生的地方。一个节目需要有一个价值导向，而奇葩说的价值导向就是没有价值导向。在这里只是有一堆奇葩，以及吸引更多的奇葩慕名而来，但这里又不鼓励和标榜奇葩，而是接受奇葩，同时接受非奇葩的人、不认同奇葩的人、认同奇葩的人，总之，这里崇尚多元价值观的碰撞与融合。

新一季第二集的辩题是“如果你奋斗的城市空气质量越来越差，你要不要离开她？”，好接地气的题目，这不就是说的北京吗？再拓展开来，其实就是那个大问号，要不要逃离北上广？媒体上大家激烈地讨论了几年也没个结果，事实是，年轻人还是更偏向于到大城市打拼而不是回家乡，尤其是刚毕业的大学生。而且辩题中大城市只有空气差这一条弊端，正方很容易从奋斗、改变这样的角度延展开来，所以这道题目本身就是偏向于正方的。于是我就期待着反方怎么样找到新的角度说服观众，果然黄执中跳出个人奋斗的范畴，上升到国家和人民层面，提出了新颖的观点：一个国家最大的骄傲，不是人们在恶劣环境中奋斗，而是培养出娇贵的人民，在意空气质量说明我们追求生活，要追求生活当然要逃离。让人转过弯来理解之后，不由得不赞同，很容易马上变换观点。

观看辩论的有趣之处就在这里，面对两个针锋相对的观点，听完一派说辞后连连点头，接着听另一派竟然又觉得非常对，以至于刚才的想法自己都会否定，可是紧接着上一派又出来揪出这个论点的漏洞，让人又开始摇摆不定。黄执中的这个观点听起来非常有理，但是再往下想会发现，整个国家和人民的事情跟我个人的具体选择又有什么关系呢？正方好几个人都开始反驳这一点，其中罗胖罗振宇的一句话点醒了我：讲话不需要负责，行动是要付出代价的。人们做选择时会考虑很多道理，但是选择完了是一定要承担相应的代价的，如果承担不起，自然就会换个选择，其实那么多分不清的道理都没用，最后的统计结果用事实说话才对嘛。

辩手们为了辩论可以从观点出发引出很多例证、道理，而换个持方同样可以说的天花乱坠，可是自己真的选择时那可真不一定了。《奇葩说》这个节目也是一样，在一个话题下可以迸发出很多观点想法，甚至可以延伸到价值观的冲突，但是最终讨论结束后并不会有一个确定的结果和道理，我本以为可以从这个节目中学到很多道理，其实道理全都是苍白无用的，得到的最好只是一些启发，而不是完全认死一个理、完全树立一种价值观。

一切理论都是灰色的，只有生命之树常青。
—歌德《浮士德》

如何能够富成一个废物

发表于 2017-03-28

只需三步：

  1. 闭上眼睛
  2. 数100只羊
  3. 在梦中富成一个废物

说正经的，这几天跟人闲聊时，大家说到钱的好处，尤其是毕业生们将要踏进社会这个看钱的地方，都幻想着赶紧挣到花不完的钱，那一定是很多钱，多到可以支撑一个废物式的生活。难怪『富成废物』成了热门词汇。

那么，一个普通人如何能够富成一个废物呢？

必须是普通人

首先，我们只讨论普通人。毕竟我只是个普通人，非普通人的样子我又不清楚。就像那个故事说的那样：两个砍柴的樵夫在讨论皇帝的生活，一个说皇帝砍柴是用的银斧头，一个说不对肯定是用的金斧头。可我们知道，皇帝根本不用砍柴嘛。

更重要的是，从普通人到富成一个废物是一个过程，这个过程恰恰是最重要的，因为没有对比就没有快乐。一个皇帝从出生那一刻可能就注定了是皇帝，他从未体会过其他人仰视财富和权利的那种渴望，那他毫不费力得来的财富并无什么意义，他一定是有其他的追求。

一个废物的样子

在讨论如何实现这个富成废物的目标之前，先来看看这个废物目标是什么样。

现代化的社会没有了固化的阶级，相对来说人们的上升渠道更广阔了，信息的不对称缓解了许多。即使是这样，普通人对于财富顶端人群的样子的刻画，来自于新闻的浮光掠影、微博上的只言片语、影视作品里的夸张渲染以及剩下的想象，这一定是有很大偏差的。以至于我们对一个富豪废物的想象可能偏离了方向。

富豪废物可能在乎的并不是钱权色，那些东西我们觉得爽可能仅仅是因为满足了想象而觉得爽，就像上一条所说，他们一定是有其他的追求，即使是废物一般什么也追求不到，但是在追求啊。

好吃懒做穷奢极欲不是富成一个废物后的状态，而可能会是一个穷废物的状态。试想，这些个欲望就是一个个的神经刺激，将来发明一种药物不断给人注射达到这个效果不就行了嘛，因为当人类继续发展物质丰富更加文明时，不会允许把穷人废物一股脑杀光，而会把他们置于虚幻之中了事。尤瓦尔赫拉利已经在《未来简史》中描绘了这样的场景，人工智能的发展提高了生产力，淘汰下的穷废物就让他们沉迷于电子游戏或者药剂好了。

关键是如何

重点来了，既然不能消极地靠撞大运，那么一个普通人能有什么办法成为一个富废物？

态度当然是要积极，只不过我们怕的是，自己再怎么拼命努力也赶不上时间，不是有句话说嘛，我奋斗了十八年才能跟你坐在一起喝咖啡。最根本的问题是，这个社会是否足够公平，上进的人的上升渠道是否变得更宽泛了？我认为是的。这个问题让我想起中学英语课上经常出现的一个题目，城市生活好还是乡村生活好？要答好这道题一定要列举出城市的各种优劣和乡村的各种优劣，可是要我这个来自农村的孩子说，那当然是城市好嘛！我们现在所在的这个国家确实还有很多很多问题，但是跟三十年前比一比如何，再跟百年前比一比如何，再跟周围几个邻居比一比呢。

前几天有报道说，连清北毕业的高材生都买不起学区房了，还公不公平啊？可是我觉得，人们可以不论背景和阶级都来靠实力竞争买房，人们只是有钱就可以买到房子，这不正说明公平吗？所以，在更加公平的环境下，更应该以符合规则的正当方式获取财富，在背景和阶级不再那么受重视的环境下，更应该注重提升自我的实力。

因此，这样已经排除了通过犯罪、投机、攀关系等方式暴富的路径。那就老老实实上班搬砖喽。然而，一个大学毕业生仅靠工资想买房，可能买得起的时候也快退休了，更别说富成个废物了。事实是，有人一直努力挣钱理财看基金没什么效果，有一天打开股票账户发现前面投的一个公司上市账户里多了数不清的零，有人整天写作教书演讲，有一天发现之前买的比特币怎么特么翻了1000倍。原来，要富成一个废物最重要的是有那个废物的运气呐。当然，运气自然不是白来的，普通人能做的应该是：

但行好事，莫问前程。静待财富来。

到这里越发有些鸡汤味了，这只是因为我并不知道具体该怎么做，毕竟我还这么穷。但我知道的是，我应当保持『天生我材必有用，千金散尽还复来』的心态，认真做好每一件真正重要的事，养成一些习惯，譬如学习。通过学习我至少知道了，富成一个废物的另一个意思是获得财富自由，财富自由指的是，不用为了生活必需出售自己的时间，这显然意味着打工无法获得财富自由，钱生钱才对，那么是不是该去学点投资？要学明白更要知道练习的重要，譬如通过练习学会把一个问题清晰地思考分析和写出来。

财富自由不是终点

关于这个问题我还想写出来的是，财富自由不是终点，也不是全部。若把这当成终点，那么等达到这个终极目标之后面对的是什么呢？一片虚无，那些想象中富成废物的种种好处将会烟消云散。从这个意义上来说，把财富自由作为终极目标更加不会获得财富自由。

正如前文所说，富废物一定有其他的追求，也许我们都该更看重些其他的东西。譬如快乐，能有大把的钱买好几套房是一种快乐，能够通过劳动自己挣到钱也是一种快乐，能为他人服务也是一种快乐，能写出好代码做出好产品更是一种快乐。

作为一个普通的年轻人，我为生活在这个物质和文化充盈的时代而感到幸运。还有那么多精彩值得去发现，还有那么多美好值得去体会，即使注定不能富成一个废物，也要保持乐观啊！

现在是不是不像原来那么想富成一个废物了？

人在行云里

发表于 2017-03-21

似乎是春天来了，空气中氤氲着蛰伏已久的悸动，津南的风还是那般无规可循，却以突如其来的面貌撩拨起青春的渴望。

那是一个明媚的午后，我和她漫步在空旷无人的路上，穿插在青青点点的坪间。脚下的块状板石一个隔着一个延伸开来，两块石板间的距离略微小了些，一步跨一节太拘谨，一步跨两节又感勉强，每走一步都忐忑着，不知是因为落脚点难找，还是因为心下慌乱难安。

她的声音很低，语速不快，轻轻的话语直挠进我的耳廓，那是一种信任的倾诉，像一汪清泉汩汩而出，又似片片雪花扑簌簌飘落。我听得出神，却全然不记得听到了什么，那个毫无头绪的故事在我耳中溜了一遍就不知所踪了。

有几次我歪过头，想看她埋藏着怎样独特的眼神，都被她的秀发遮拦，一丝一缕的线条排布开，张开恰到好处的角度，又顺流而下撇出精巧的弧度，仿佛层叠着的幕帘荡漾开来，掩着神秘的美感。我正随着幕帘摇摆，这时一阵风吹过，一下子吹散了所有的神秘，只留下一片凌乱，就像银河拨开面纱浮现出错落的星点。

我回过神来晃了晃脑袋，眼见路边多了几树桃花，她已经先一步蹦了过去，映照着零星几朵刚开的桃花，我终于看到了她的微笑。她大概是又复原了欢脱的状态，我知道刚才的阴霾已经散去，她需要的就是倾听。

顿时，我感觉天空更加明朗，阳光更加温暖。望向远处，刚好一架飞机从云层中窜出。看，飞机飞得好低啊。你坐过飞机吗？没有呢。我也没坐过，不知在云中是什么感觉。

我呆呆地望着天边的云，飘忽不定又实实在在的云，怎么看也看不厌，只觉得云是那么美好的存在。她问我在想什么，我说我想到了一首诗：

天上有行云，人在行云里。
此处有春风，十里不如你。

还没等她应答，我发现我从梦中醒来。

想必是春天来了。

虚妄之念

发表于 2017-03-12

很久没有更新了，每次想要坐下来写东西都不能完全静下来，因为找实习面试的事情，心中常常感到焦虑不安。前天经历了我的第一次正式的面试，终于感到舒畅了许多。

很多时候就是这样，在一些事情抉择面前诚惶诚恐，一旦真正到了那个时候，发现也就是那么回事。于是我总是尽量保持淡定，不过虽然表面平静如水，其实是火气攻心，厌食难眠。所以说内心强大真不是说说那么简单，我能做到的只是控制自己的情绪，但要真的做到控制自己的心态和精神还是很难的。

那么能做的就是多多反思总结了，这一周多的焦虑让我感到疲惫，现在想来焦虑的原因，最大的一条就是来自于心底的妄念。

我想要应聘一份暑期实习，首先要搞清楚对我而言最重要的是什么。不是高薪水，一个实习生不应该以薪水为导向，实际上我认为即使是第一份正式工作也不该追求高薪水，这个权重被大多数人高估了；也不是城市，城市决定了地理位置，离家里/学校的远近只影响通勤费用的多少，想要为以后在哪里扎根着想也有些多虑；也不是公司，大小公司各有优缺点，认同某个公司的产品并不表示非要去那个公司，尤其在跳槽相对频繁的互联网行业。

最重要的，应当是工作的方向和内容，因为具体做什么工作才决定了未来的道路和前景，而一个理智的人的选择应当是面向未来的。

问题就出在这里，当我在考虑做什么工作时，我是在想象未来会是什么样子。可是既然是想象，那真是能想出数不尽的样子，人总是会在这时产生错觉。我打开大公司的招聘网站挨着看每一个岗位的职位要求，不知哪来的自信总让我觉得每一条我都能扯上点关系。我做过某某项目，这一条肯定没问题；我学过那个课程，这一条也行；这一条里说的，别人也总这样评价我……于是还没有冷静分析就开始浮想联翩。才起于毫末处，便妄想九层台。

某一天我看到一个词『存量』，我知道这其实是因为我太在乎存量了。我害怕积累的那点微薄的存量会派不上用场，害怕存量变成了沉没成本，人总是会受到沉没成本的牵绊。对于一个终身学习者，存量反而成了一个包袱，本来重量也不大，可因为一直背着不愿甩掉而变得异常沉重。在做选择时，去除存量也就能消除大部分妄念，这应当是更为正确的态度。

好了，甩掉包袱一身轻松，再来进行选择，这时考虑最多的当然是兴趣，我到底喜欢什么样的工作？我到底喜欢什么人？以前的我会打破沙锅问到底。可是问了太多遍之后我明白了，这是问不出结果的。我真的能告诉自己我最热爱什么最想要什么吗？当我遵照这个答案会变得最快乐最幸福吗？不可能的。人生没有彼岸，问题也没有答案。

陷入这种诘问中也就是陷入了虚妄。稍微观察一下周围就能发现，有太多人都有这样的妄念。以为有了什么就有了一切，以为到了哪里就到了重点。殊不知，九层台之上还有下一个九层台。

倒不如，踏踏实实踩上第一节台阶。也只有当我真正经历了面试，才更多明白这个岗位需要哪些技能，会面临哪些挑战，还需要多做什么准备。这时才发现，原先的妄念，在真真切切的现实空气里本应该消散。

其实，应该学习天真无邪的孩童，为一个小小的玩具争得又哭又闹，毫不在意那些遥远的看不到的东西。以及那些阅尽沧桑的老人，经历过轻和重，然后知轻重，经历过得和失，然后明得失，不陷入虚妄之中，又能不逾钜。

Running Pregel+ Programs

发表于 2017-02-27

Download

  • Download system code and extract to $HOME/pregelplus

  • Download application code of Hash-Min and extract to $HOME/hashmin

  • Download toy graph and put it onto HDFS

    hadoop fs -mkdir /toyFolder

    hadoop fs -put toy.txt /toyFolder

Compilation

  • Download makefile sample and put it under $HOME/hashmin

  • replace the system code directory of the makefile

  • compile the source code to the binary file run

    make -f Makefile.txt

Running

mpiexec -n number-of-processes -f process-to-machine-mapping-file compiled-binary-file other-arguments
