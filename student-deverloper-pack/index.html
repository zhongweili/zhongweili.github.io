Student Deverloper Pack

发表于 2018-04-13

近来开始多多使用 github（我的 github：zhongweili），在考虑要不要开通付费版，就可以创建私有仓库了，恰好学长推荐了 github 学生账号，原来学生账号竟有无限量私有仓库数，果断申请开通。

必备条件

  • 教育网邮箱（.edu.cn 结尾的域名邮箱）

也有网友提到，某些没有 edu 邮箱的学生可以提供学号，总之必须是未毕业大学生

开通步骤

  • 访问 https://education.github.com/
  • github 账户绑定 edu 邮箱
  • 按步骤申请，填写个人信息（注意填上 edu 邮箱），写明申请理由
  • 等待批准，几分钟就收到了回复
  • Get it！
