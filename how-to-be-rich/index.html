如何能够富成一个废物

发表于 2017-03-28

只需三步：

  1. 闭上眼睛
  2. 数100只羊
  3. 在梦中富成一个废物

说正经的，这几天跟人闲聊时，大家说到钱的好处，尤其是毕业生们将要踏进社会这个看钱的地方，都幻想着赶紧挣到花不完的钱，那一定是很多钱，多到可以支撑一个废物式的生活。难怪『富成废物』成了热门词汇。

那么，一个普通人如何能够富成一个废物呢？

必须是普通人

首先，我们只讨论普通人。毕竟我只是个普通人，非普通人的样子我又不清楚。就像那个故事说的那样：两个砍柴的樵夫在讨论皇帝的生活，一个说皇帝砍柴是用的银斧头，一个说不对肯定是用的金斧头。可我们知道，皇帝根本不用砍柴嘛。

更重要的是，从普通人到富成一个废物是一个过程，这个过程恰恰是最重要的，因为没有对比就没有快乐。一个皇帝从出生那一刻可能就注定了是皇帝，他从未体会过其他人仰视财富和权利的那种渴望，那他毫不费力得来的财富并无什么意义，他一定是有其他的追求。

一个废物的样子

在讨论如何实现这个富成废物的目标之前，先来看看这个废物目标是什么样。

现代化的社会没有了固化的阶级，相对来说人们的上升渠道更广阔了，信息的不对称缓解了许多。即使是这样，普通人对于财富顶端人群的样子的刻画，来自于新闻的浮光掠影、微博上的只言片语、影视作品里的夸张渲染以及剩下的想象，这一定是有很大偏差的。以至于我们对一个富豪废物的想象可能偏离了方向。

富豪废物可能在乎的并不是钱权色，那些东西我们觉得爽可能仅仅是因为满足了想象而觉得爽，就像上一条所说，他们一定是有其他的追求，即使是废物一般什么也追求不到，但是在追求啊。

好吃懒做穷奢极欲不是富成一个废物后的状态，而可能会是一个穷废物的状态。试想，这些个欲望就是一个个的神经刺激，将来发明一种药物不断给人注射达到这个效果不就行了嘛，因为当人类继续发展物质丰富更加文明时，不会允许把穷人废物一股脑杀光，而会把他们置于虚幻之中了事。尤瓦尔赫拉利已经在《未来简史》中描绘了这样的场景，人工智能的发展提高了生产力，淘汰下的穷废物就让他们沉迷于电子游戏或者药剂好了。

关键是如何

重点来了，既然不能消极地靠撞大运，那么一个普通人能有什么办法成为一个富废物？

态度当然是要积极，只不过我们怕的是，自己再怎么拼命努力也赶不上时间，不是有句话说嘛，我奋斗了十八年才能跟你坐在一起喝咖啡。最根本的问题是，这个社会是否足够公平，上进的人的上升渠道是否变得更宽泛了？我认为是的。这个问题让我想起中学英语课上经常出现的一个题目，城市生活好还是乡村生活好？要答好这道题一定要列举出城市的各种优劣和乡村的各种优劣，可是要我这个来自农村的孩子说，那当然是城市好嘛！我们现在所在的这个国家确实还有很多很多问题，但是跟三十年前比一比如何，再跟百年前比一比如何，再跟周围几个邻居比一比呢。

前几天有报道说，连清北毕业的高材生都买不起学区房了，还公不公平啊？可是我觉得，人们可以不论背景和阶级都来靠实力竞争买房，人们只是有钱就可以买到房子，这不正说明公平吗？所以，在更加公平的环境下，更应该以符合规则的正当方式获取财富，在背景和阶级不再那么受重视的环境下，更应该注重提升自我的实力。

因此，这样已经排除了通过犯罪、投机、攀关系等方式暴富的路径。那就老老实实上班搬砖喽。然而，一个大学毕业生仅靠工资想买房，可能买得起的时候也快退休了，更别说富成个废物了。事实是，有人一直努力挣钱理财看基金没什么效果，有一天打开股票账户发现前面投的一个公司上市账户里多了数不清的零，有人整天写作教书演讲，有一天发现之前买的比特币怎么特么翻了1000倍。原来，要富成一个废物最重要的是有那个废物的运气呐。当然，运气自然不是白来的，普通人能做的应该是：

但行好事，莫问前程。静待财富来。

到这里越发有些鸡汤味了，这只是因为我并不知道具体该怎么做，毕竟我还这么穷。但我知道的是，我应当保持『天生我材必有用，千金散尽还复来』的心态，认真做好每一件真正重要的事，养成一些习惯，譬如学习。通过学习我至少知道了，富成一个废物的另一个意思是获得财富自由，财富自由指的是，不用为了生活必需出售自己的时间，这显然意味着打工无法获得财富自由，钱生钱才对，那么是不是该去学点投资？要学明白更要知道练习的重要，譬如通过练习学会把一个问题清晰地思考分析和写出来。

财富自由不是终点

关于这个问题我还想写出来的是，财富自由不是终点，也不是全部。若把这当成终点，那么等达到这个终极目标之后面对的是什么呢？一片虚无，那些想象中富成废物的种种好处将会烟消云散。从这个意义上来说，把财富自由作为终极目标更加不会获得财富自由。

正如前文所说，富废物一定有其他的追求，也许我们都该更看重些其他的东西。譬如快乐，能有大把的钱买好几套房是一种快乐，能够通过劳动自己挣到钱也是一种快乐，能为他人服务也是一种快乐，能写出好代码做出好产品更是一种快乐。

作为一个普通的年轻人，我为生活在这个物质和文化充盈的时代而感到幸运。还有那么多精彩值得去发现，还有那么多美好值得去体会，即使注定不能富成一个废物，也要保持乐观啊！

现在是不是不像原来那么想富成一个废物了？
