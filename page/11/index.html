宝贵的注意力

发表于 2016-08-15

一夜之间，王宝强这个可爱的宝宝吸引了几乎所有人的注意力，仿佛身边的朋友都成了最熟悉王宝强的人，大家都议论纷纷，发表着自己的观点。为何会这样？

这得益于移动互联网对信息传播的发酵，互联网加速了传播的过程，也放大了传播的信息。朋友圈里出现了各种截图和段子，公众号里则是各种观点的争论碰撞，甚至工作交流圈子里也有一些讨论，至于微博，当然是炸了一整天。很多人旗帜鲜明地支持王宝强，表现出强烈的愤怒情绪，有些人试图以温和的态度理解肇事者，还有的人借此调侃某度的百科某斯的文案，也有人想要阐明这是家庭事件不是社会事件，后来又有人指出那则声明何其糟糕……各种观点层出不穷，关于不同的观点、持不同观点的人又引发新一轮的议论，真是充分体现了人们的认知盈余。

算了吧，说认知盈余太宽泛了，其实这反映了人们注意力的转移。在新的信息传播环境下，每个人接触到的信息太多也太快，这很容易占有我们宝贵的注意力。譬如我从朋友圈看到很多人在聊Wings、CN dota， 好奇的我也马上去看了直播，我在朋友圈经常看到佐藤胜利这个日本歌手，慢慢的我对这几个字特别敏感和熟悉，看到他的歌他的信息也不自觉关注一点点。好奇本没有错，只是凡事都有个度，如果总是被热点事件、新鲜资讯所吸引，那么时间将会缩水，因为注意力已经涣散。

本来我对注意力的认识也就是一个影响工作效率的存在，正好最近看了李笑来在“通往财富自由之路”里面的深刻教导，才意识到注意力是大于时间大于金钱的存在。很简单的道理，注意力是我们内在的选择，它不像时间是游走在我们身体之外的东西，而是可以通过我们的控制完全为我们服务的，所以注意力是比时间更宝贵的财富。

同时，笑来老师还指出了人生中浪费生命、虚掷光阴、永陷贫困的三个大坑：莫名其妙地凑热闹，心急火燎地随大流，操碎了别人的心肝。引以为戒。重新理解了注意力的宝贵，那如何更好的使用它呢？笑来老师的三个建议是:首先，要把注意力放在自己的成长上;其次，要放在自己的真爱上;最后，把注意力放在对整个社会真正有贡献的事情上去。

这些金科玉律都对吗？或者这些应该当做金科玉律吗？也未可知。放弃玩乐而疲于奔命一定会成长与成功吗？成功就意味着快乐吗？必须一点都不浪费宝贵的注意力吗？于人生而言浪费一定是可耻的吗？带着这些思考我会再写一篇文章。

科学上网原理

发表于 2016-07-22

由于GFW的存在，『科学上网』已经几乎成了一个专有词汇，明白其含义的人大概早就在科学地上网了，不过这背后的原理可能还比较模糊，常常听到的『翻墙』、『VPN』、『Hosts文件』等词汇都是什么含义呢？在此试着利用所学知识进行梳理。

首先要知道的是，科学上网的原理和方法不是一成不变的，因为围绕着GFW的攻防战一直在持续不断地进行着，新的突破防火墙的技术手段层出不穷地涌现出来，防火墙也在不断地进化以识别新的方法。

GFW最基础的方法是连接重置，通过检测TCP连接的IP包中的非法关键字识别出目标连接，然后伪造RST包发送给TCP连接两端，使连接中断。这时只需要使用https加密方式就可以绕过墙。实际上所有的翻墙手段都对数据经过了加密处理。

更凌厉的方法是IP/端口封锁，部署在骨干路由器上，可以简单有效地彻底封锁目标IP。应对的方法是使用代理服务器，通过安全的代理服务器来间接进行访问，达到绕过IP封锁的目的。

还有一种方法是DNS污染，用户在访问网站时都要经过DNS解析，利用这一点，在用户进行特定网站的DNS查询时，通过抢先发送伪造的DNS结果，导致DNS解析到虚假的IP地址从而阻止用户访问非法资源。有一种修改HOSTS文件的策略就可以应对这一招，因为用户在进行DNS查询之前会预先查找HOSTS文件的映射内容，不过这个方法成功的前提是映射的IP没有被封锁。

针对以上的几种方法，比较稳定的科学上网方式是使用VPN，也就是通过虚拟服务器，使用特定的加密协议，间接访问封锁的资源。但VPN也是有破绽的，很多加密连接在握手时，发送的数据包有一定特征，因为加密连接在设计时优先保证数据不可被解读和伪造，很难剔除掉特征，GFW利用这一点，通过分析特定加密协议的特征判断出非法的握手包，然后阻断连接，因此使用VPN进行翻墙也不是万全之策。

前几年还流行过一段时间GoAgent，这是依托于Google云服务GAE这种特殊代理的手段，现在已经完全不能用了。

目前比较稳定可靠的翻墙方式是Shadowsocks，这是专门为了翻墙而设计的一个项目，不同于之前代理服务端的方式，shadowsocks增加了代理客户端，它依托socks代理协议，在代理的两端之间对内容进行加密，加密的方式是预共享秘钥，没有明显的握手包，因此很难被发现。除非使用人数太多，目标太大则可能暴露。

关于科学上网的具体方法，暂且按下不表。

小粉红的爱国

发表于 2016-07-16

最近网上出现了一大片的『小粉红』，我专门去百度了这个词的涵义，原来小粉红这个称呼早就有了，进入微博时代之后，波及的范围越来越广，渐渐演变成一个偏贬义的『网络爱国青年』。

本来我对这种事熟视无睹，也就是微博这个大广场上又喧闹了些而已，直到我看到咪蒙发的一篇瞬间十万+的《永远爱国，永远热泪盈眶》，我惊讶于这样毫无逻辑的文字竟然那么振振有词，我惊讶于这样情绪化的表达竟然传播得那么迅猛。『精英』与『庸众』的分庭抗礼本是一种平衡的常态，可是聚集起来的『庸众』实在可怕之极。从抵制菲律宾香蕉，到围猎赵薇、戴立忍，小粉红抱起团来行动力惊人，细思级恐。

何为爱国？如何爱国？我想起学者熊培云在《重现发现社会》中的阐释：
归根到底，国家只是全体国民缔约产生的一个组织，是国民用以谋求幸福生活的工具。在此意义上，爱过主义者真正要做的不是忙于督促每个人去爱国，而是要让国家能够爱每一个人。

有人说小粉红的极端无可厚非，他们只是缺乏表达的渠。这是一个好角度，人需要表达，可是现在没有畅通的渠道，却有着一呼百应的引导，这不仅包括咪蒙这种大V，还有强制传播『一点读不能少』的官媒。

一个现代国家的价值内涵是什么？国民应该就此秉持一种怎样的情感？一个国家有着怎样的未来，必首先决定于全体国民尤其是年轻一代对国家抱持一种怎样的态度。

也来谈谈豌豆荚

发表于 2016-07-05

上周末就在传闻豌豆荚的各种消息，今天豌豆荚正式被阿里巴巴2亿美元收购了，很多人在替王俊煜惋惜，因为豌豆荚几年前还被估值10亿美元，现在这个价钱可是缩水了不少。然而不可否认的事实是，这个80后年轻创始人王俊煜用了6年时间打造了一个2亿美元的公司，这一点有几人能做到？

我很早就关注着王俊煜的博客『猫窝』，因为他很喜欢猫，并且自诩为猫，还听说他的公司办公室里就养着好多只猫。受猫的世界观的影响，我感到我领略到了互联网上的大部分令人激动的东西，自由、情怀、理想、价值……我觉得豌豆荚就是中国互联网上一个独特的存在。

因为我更早就一直用着豌豆荚APP，自从上大学第一次使用智能手机开始。以至于每次换手机或者重装系统后第一个下载的应用就是豌豆荚，确实，作为一类发现应用的应用，应用分发软件在几年前就表现出极大的前景，各个应用商店APP或者站队大公司，或者捆绑到手机厂商，而唯有豌豆荚是一个独立且宣布自己独立的存在。自然，相较于其他应用商店豌豆荚内没有那些烦人的广告和推荐，也没有那些友商的榜单，再加上豌豆荚独特的气质，一时间成为了Android应用分发渠道的一股清流，很快就做到了上亿用户和应用资源第一。

然而，豌豆荚活的并不舒坦，因为在商业上盈利永远是最重要的。主流应用市场渐渐趋于饱和，依托于微信这类大生态的微应用慢慢崛起，在Android应用的大环境下，豌豆荚需要不断找出新的方向。其实王俊煜和他的团队也确实一直在探索新的东西，其出品的『开眼』和『豌豆荚一览』就都是优质的新产品。在猫窝上王俊煜也经常分享出新的idea，关于趣味的、关于好内容的、关于新形式的。

但始终都挡不住，商业的洪流和资本的巨浪。在金钱社会里，一直都是残酷和冷血的。

今年上半年，就连最不着急的知乎都开始急着赚钱了，频频推出各种变现模式的尝试。这其中或许都有无奈吧。下午王俊煜在猫窝上更新了一篇文章，无奈之外，依然有着那份锐意进取的态度，依然追求着他的价值观。豌豆荚的前景会是怎样呢？我想到了魅族，之前小而美的魅族和flyme，在加入阿里后一年之内变化了太多太多，魅族确实不再那么小而美那么有个性了，但魅族这两个字现在太有分量了，如果没有阿里的入股，可能早已死在智能手机的乱军混战中，你说这值还是不值呢？或许这就是商业的力量和趋势？

虽然我个人并不希望豌豆荚加入阿里巴巴，也不知道还会不会喜欢豌豆荚，不过我喜欢王俊煜做事的态度：但行好事，莫问前程。

Windows装机过程

发表于 2016-06-24

电脑使用久了，难免会遇到各种难以处理的小bug，再看看一团糟的文件结构，这时总会想重装个系统，一切重新开始，纵享丝滑。

很多人会觉得装系统复杂且麻烦，其实现在windows10系统的安装\升级\还原过程已经变得越来越简洁而高效，无需借助捆绑广告病毒的工具，也无需繁琐的步骤，只要几次点击操作，等待半小时就可完成一次系统的新生。因此装系统并不可怕，可怕的是重新安装需要的软件和搭建熟悉的环境。在这里记录下本次装机过程，以备他用。

系统及驱动

windows10系统的安装过程已经很接近傻瓜式安装了，以前装个系统需要用系统光盘，从光驱加载，聪明的会用老毛桃、大白菜之类的工具修改带有系统文件的U盘，然后从U盘加载，再设置引导，最简单的也要用一键ghosts这样的工具从硬盘直接安装。而安装windows10系统，只需下载iso镜像文件，打开点击setup.exe，跟着提示一路点击确定就搞定了。伴随着多次电脑重启，连带着把各种驱动也都装好了，简单至极。

网络和浏览器

进入全新的系统，首先要做的当然是连接上网络。如果是固定IP上网的，需要设定电脑的IP地址及DNS地址，这些最好在重装系统之前先记下来。测试网络情况当然要进去浏览器查看，虽然windows10有了革新的Edge浏览器，但我还是只认Chrome，安装Chrome需要临时翻墙，这时下载一个PC版Lantern即可开始安装，装好后打开Chrome登陆Google账号，马上开始同步我的收藏夹、书签、历史纪录、扩展应用，很快地址栏旁边出现了我熟悉的Adblock、SwitchyOmega、Pocket，与之前的使用别无二致。这就是我使用Chrome的原因之一，这也是浏览器可以作为操作系统的原因。

效率工具

接下来安装几个我喜欢的工具

  • Wox 在OSX下用习惯了Alfred，这个工具便是windows下的替代品
  • Everying 这是一个文件索引工具，将所有文件进行一次索引，任意进行搜索，也在Wox中支持
  • Rolan 这是一个应用和文件的快速启动器，有了它，桌面可以变得一干二净
  • f.lux 根据日出日落周期调节屏幕色调，护眼

重要软件

下面是重头戏，各种软件的安装

  • Office2016 毕竟哪里都离不开office，装高版本，记得激活
  • Acrobat Pro windows没有提供pdf查看器，所以这个也是必须的
  • WeChat 微信实在太普及，生活与工作很难分开，PC版微信也要用
  • QQ轻聊版 不要安装原版QQ，实在臃肿，轻聊版还不错
  • 网易云音乐 喜欢歌单和个性化推荐
  • utorrent 最著名的BT下载工具
  • Markdownpad 2 正在用的这款Markdown编辑器
  • 有道云笔记 为了使用有道云协作而装
  • winrar 老牌压缩解压缩软件
  • 百度云管家 常常需要从百度云下载东西，稍微大一点的文件需要通过云管家下载
  • CAJViewer 看论文需要
  • 其他 photoshop、钉钉、印象笔记、xmind、Listary

开发环境

更重要的是开发环境的安装和配置

  • sublime text 个人喜好的编辑器
  • notepad++ 看某些奇怪格式的文件会需要这个
  • java java的使用太普遍了，下载jdk、安装jre、配置JAVA_HOME
  • visual studio windows必备
  • git 版本控制与团队协作
  • vmvare vSphere Client 服务器和虚拟机管理
  • wireshark 网络抓包
  • matlab 可以通过参加kaggle比赛，向MathWork申请正版Matlab

随着技能树的扩充，这一部分还要不断更新

为好内容付费

发表于 2016-06-08

在做了一番思想斗争后，我订阅了和菜头在『得到』APP里面开设的《槽边往事》付费专栏，我使用了微信支付，点个按钮就从APP跳转到了微信支付的页面，输个密码就完成了订阅，同时也就扣除了199块大洋，然后就回到『得到』APP可以看订阅的文章了。

而在此之前，我都是在『槽边往事』微信公众号里面免费阅读和菜头的文章，时常感受到他的幽默、犀利和宽厚。和菜头是一个已经在互联网上写作近20年的胖子，他像一个魔法师一样总是能变化出令人称奇的作品，且产出颇丰。但影响我最深的不只是他层出不穷点子和笔耕不缀的坚持，而是他坦坦荡荡的态度以及践行的勇气。这一次，只不过是把他重复过很多次的那个观点用大白话说给我们听并且展示给我们看：唯有收费，才能真正衡量一样事物的价值。
一样事物的价值用钱来衡量当然是天经地义的，只是在互联网的大环境下我们早就习惯了免费，因为互联网上的信息天生就是免费的。互联网公司创造了许多划时代的商业模式，网络上的信息定然有成本，也可以通过广告商、会员制、粉丝经济等把成本转移出去，让用户接触到免费的信息。可是情况一直悄悄在改变，信息越来越多，越来越杂，更多的人感受到了信息过载。于是我们才发现，信息免费，但时间有价，筛选信息其实也就是为信息付费。

前一段时间，一些付费的模式就已经开始蠢蠢欲动了，大弓、值乎、分答先后开始尝试所谓『知识收费』，引起了一片波澜，许多人开始接受为知识付费。其实知识不就是好的内容吗？或许，为好内容付费是一个更大的趋势。微信订阅号的更新过程就是一个缩影，增加原创标签、完善举报规则、加入文章打赏按钮，不断地在重视高质量内容，简书、头条号也开始有这种打赏机制。打赏还是用户自愿的，『得到』做的更彻底，强制付费，我很早就有安装这个APP，上面的付费内容一直都在慢慢增多，只是今天我刚找到我信任的寄托。这些是文字的内容，就在前天6月6日，『喜马拉雅』APP第一次推出了付费音频《好好说话》，如果成功，一定还会推出更多。

和菜头说：当为内容付费的行为开始的时候，我们终究会因为个人行为改变整个网络世界的习惯，并且，为创生更多优质内容和服务，奠定了珍贵的基础。这很难。难又如何？我在思想斗争之后选择了支持。
